# Task.go
> a simple project about the `ToDo` and run it by one command !
* `go` v1.14.5
* `gin` v1.6.3
* `gorm` v1.20.1
* `gopkg` v1.62.0
* `sqlite3` v3.33.0

## How To Run
### build
```shell script
go build
```

### run
```shell script
# Mac / Unix
./task conf/config.ini

# Windows
task.exe conf/config.ini
```

finally, open your browser and input the url : *http://127.0.0.1:9000/*
